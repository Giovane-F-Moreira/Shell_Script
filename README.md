<h2>Scripts tested on:<br> <h3>ROS Melodic, Gazebo 9, Ubuntu 18.04</h3> </h2>

Before running the scripts install the **Dialog** and execute permission


``` 
sudo apt-get install dialog
chmod a+x -v *.sh
cd ROS_Workspace
./menu.sh
```
